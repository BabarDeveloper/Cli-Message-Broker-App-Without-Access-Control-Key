# CLI Message Broker

A lightweight, database-free message broker using Node.js and Socket.io.

## Features

- ✅ Create topics as a publisher
- ✅ Subscribe to topics
- ✅ Real-time message delivery to all subscribers in a group
- ✅ No database - all in-memory
- ✅ Pure CLI interface
- ✅ Multiple topics support
- ✅ Automatic cleanup on disconnect

## Installation

```bash
# Clone or create project directory
mkdir cli-message-broker
cd cli-message-broker
